【数论】莫比乌斯反演

一、数论函数 积性函数

数论函数指定义域为 \(N*\) 的 函数。数论函数也可以视作一个数列。

积性函数的定义

若函数 \(f(n)\) 满足 \(f(1) = 1\) 且 对于任意正整数 \(x, y\) 满足 \(\gcd(x, y) = 1\)， 都有 \(f(xy)\) = \(f(x)f(y)\)，则称 \(f(n)\) 是一个积性函数。若没有互素条件也满足 \(f(xy) = f(x)f(y)\)，则称 \(f(n)\) 是一个 完全积性函数。

积性函数的性质

若 \(f(x)\) 与 \(g(x)\) 都为积性函数，则以下函数也为积性函数：

  • \[h(x) = f(x^p)\]

  • \(h(x) = f^p(x)\)

  • \(h(x) = f(x)g(x)\)

  • \(h(x) = \sum_{d|x} f(d)g(\frac{x}{d})\)

常见的积性函数

  • 单位函数：\(\varepsilon(n) = [n = 1]\) （完全）
  • 恒等函数：\(\text{id}(n) = n\) （完全）
  • 常函数：\(1(n) = 1\) （完全）
  • 欧拉函数：\(\varphi(n) = \sum_{i = 1}^n [\gcd(i, n) = 1]\)
  • 莫比乌斯函数：\[\mu(n) =\begin{cases} 0 & \exists d > 1, d^2 | n \\ (-1)^{\omega(n)} & \text{otherwise.} \end{cases} \] 其中，\(\omega(n)\) 表示 \(n\) 的本质不同质因子个数。它是一个加性函数（即任意 \(\gcd(a, b) = 1\)，满足 \(\omega(ab) = \omega(a) + \omega(b)\)。
